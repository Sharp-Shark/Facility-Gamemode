ID Cards

Lets people go places (depending on its access tier).

ID Cards are a very useful and common item. Simple plastic devices with a small chip - far from sci-fi.
The JET Device (which in game is the same as an ID Card) however is state of the art technology that lets the terrorists hack their way in.
Don't ask why it doesn't have max admin tier though...

Table of ID Card access tiers:
Access Table Admin Tier 0 Admin Tier 1 Admin Tier 2 Admin Tier 3
Weapons Tier 0 Inmate Staff none Overseer
Weapons Tier 1 none Enforcer none none
Weapons Tier 2 none none Elite Guard none
Weapons Tier 3 none none JET MERCS

Admin 1 allows access to Shafts, Tram 512, Offices and Medbay.
Admin 2 allows access to Tram 256 and 128 (which lead to the exit).
Admin 3 allows access to R&D rooms with experimental technology.
Weaponry tiers allow access to armories (first A+Wallmounted, then B and finally C), the higher the tier, the better the loot.
One exception to this is Tram 384, which requires Weaponry 1.
Monsters can use all trams and regular doors, and although they can't open secure doors, they can break them.
